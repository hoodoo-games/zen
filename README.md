## About


**Every game is unique.** We believe that engine design is inherently a trade-off between batteries-included functionality and adaptability. Sometimes it's better to build a 
solution that works well for you, rather than wrangle a solution designed by somebody else. Zen aims to provide a simple yet powerful foundation
without making too many assumptions about the games that will build upon it.

**The web is the future.** We believe in the potential of the web as a univeral platform. Web-based distribution and portability are unmatched and
free from the walled gardens that dominate native software today. Modern web technologies like WebAssembly and WebGPU greatly expand the
capabilites of the platform and will gradually usher in a new age of high-performance interactive software.

Development should be fun, not frustrating.

<div>&nbsp</div>

## Development
Zen is written in Zig 0.13 targeting WebAssembly. Zen uses a thin layer of JavaScript to access browser APIs.

Zen is managed with Zig build. Refer to the `makefile` for common commands or run Zig build directly.

Zen does not have any third party dependencies.
